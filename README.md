## 📱 Что умеет бот

| Функция | Описание |
|---------|----------|
| 🧠 **Умное распознавание** | ИИ (Google Gemini) читает чек и автоматически находит все позиции |
| 📦 **Поштучное разделение** | "Тартар 4 шт" → 4 отдельные кнопки, каждую можно отдать разному человеку |
| 👥 **Любое количество людей** | Разделите чек на 2, 5 или 15 человек |
| 💰 **Автоматический подсчёт** | Бот сам посчитает сумму для каждого |
| ✅ **Простой интерфейс** | Инлайн-кнопки — всё в одном чате |
| 🔒 **Безопасно** | Ваши данные не хранятся, токены в переменных окружения |

### Пошаговая инструкция

**1. Отправьте боту фото чека**
- Сфотографируйте чётко, при хорошем освещении
- Чек должен быть полностью в кадре

**2. Дождитесь распознавания**
- Бот покажет все найденные позиции
- Проверит общую сумму

**3. Введите количество человек**
- На скольких делим чек (от 2 до 15)

**4. Для каждого человека выбирайте товары**
- Просто нажимайте на кнопки
- Можно выбрать несколько товаров для одного человека

**5. Нажмите "Подтвердить эту часть"**
- Бот покажет итог для каждого человека
- Можно начать новый чек

❓ Часто задаваемые вопросы
Q: Бот не распознаёт мой чек
A: Попробуйте сфотографировать чётче, при хорошем освещении. Чек должен быть полностью в кадре.

Q: Бот не отвечает
A: На бесплатном тарифе Railway бот может "засыпать". Отправьте /start и подождите 10-15 секунд.

Q: Можно добавить больше 15 человек?
A: Да, измените проверку в коде if not (2 <= num <= 15).

Q: Как изменить валюту?
A: В коде замените ₽ на нужный символ.
